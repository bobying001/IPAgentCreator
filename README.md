# NLPtutor: 通用 IP 数字化资产工厂 (Universal IP Factory)

本项目致力于将专家的非结构化知识（著作、讲义、笔记）全自动转化为原子化的 AI 知识库与数字化人格手册。

## 🎯 核心目标
通过“配置化”与“AI 自动协议生成”技术，让任何领域的专家 IP 内容都能在几分钟内转化为可投喂给 Google Gems 或 GPTs 的高质量资产包。

## 📁 核心项目：[02_Universal_IP_Platform](./02_Universal_IP_Platform)
目前项目聚焦于通用 IP 数字化工厂的开发，主要包含以下自动化环节：

1.  **AI 自动画像**：自动识别博主领域、价值观、受众及平台特征。
2.  **智能洗地**：自动匹配并清除特定平台的社交媒体杂讯。
3.  **协议自炼**：自动生成领域专用的萃取标准。
4.  **灵魂唤醒**：自动撰写 IP 数字化人格实操手册（System Instruction）。

## 🚀 快速启动
请进入 `02_Universal_IP_Platform` 文件夹，将原始文档放入 `source_documents` 目录，然后运行：
```powershell
python start_factory.py
```
