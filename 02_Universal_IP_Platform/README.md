# 02: 通用 IP 数字化资产工厂 (Universal IP Factory)

本项目是一个全自动化的专家知识萃取流水线。只需输入专家的原始文档，系统即可自动识别其性格特征、清洗杂讯、生成提取协议，并最终炼制成符合智能体（Gems/GPTs）投喂标准的原子化知识库。

## 🚀 核心流程

系统采用 **“六位一体”** 的自动化流水线设计，由 `start_factory.py` 统筹调度：

1.  **Step 1: 格式转换 (`extractor.py`)**  
    将 `source_documents/` 下的 PDF、Word 或 TXT 文件统一转为标准文本，存入 `raw_text/`。
2.  **Step 2: AI 自动画像 (`config_generator.py`)**  
    AI 深度扫描语料样本，自动识别 IP 姓名、领域、价值观及平台来源（如微信、小红书），并动态生成配置文件。
3.  **Step 3: 智能洗地 (`cleaner.py`)**  
    根据画像识别的平台特征，自动挂载黑名单库，剔除广告、页码、二维码等社交媒体杂讯。
4.  **Step 4: 协议炼制 (`protocol_gen.py`)**  
    根据 IP 的知识结构特征（理论型/技巧型/案例型），自动生成一套领域专用的《知识原子化萃取协议》。
5.  **Step 5: 深度萃取 (`factory_core.py`)**  
    核心引擎根据协议，将原始语料转化为高密度的原子化 Markdown 片段，并生成全局索引。
6.  **Step 6: 灵魂唤醒 (`persona_gen.py`)**  
    自动撰写一份《IP 数字化人格实操手册》，作为智能体的系统指令（System Prompt）。

## 📁 目录指南

*   `source_documents/`: **【入口】** 请在此放入该 IP 的原始著作、文章、讲义或笔记。
*   `raw_text/`: 转换后的原始文本（中间产物）。
*   `cleaned_text/`: 洗净后的纯净语料（中间产物）。
*   `generated_protocols/`: AI 为该 IP 动态生成的萃取协议。
*   `../output_assets/`: **【出口】** 最终生成的数字化资产包将按 IP 姓名在此分类存放。

## 🛠 如何使用

1.  **准备语料**：将该 IP 的文档放入 `source_documents/` 文件夹。
2.  **一键启动**：
    ```powershell
    python start_factory.py
    ```
3.  **获取结果**：运行结束后，前往项目根目录下的 `output_assets` 文件夹，获取该 IP 的知识库与人格手册。

## 💎 核心优势
- **全自动化**：从原始文件到智能体指令，无需手动编写任何 Prompt 或配置文件。
- **高保真还原**：通过“1:1 无损还原区”确保专家金句不被扭曲，通过“逻辑压缩”确保效率。
- **跨平台适配**：内置微信、知乎、小红书等多维去噪引擎。
