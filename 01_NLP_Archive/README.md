# NLPtutor: 垂直领域专家 IP 数字化转型与智能体训练系统

本项目旨在解决将特定领域专家（如李中莹老师 NLP 体系）的非结构化知识（大部头著作、教学讲义、实操笔记）转化为**高品质、原子化、AI 深度理解**的结构化知识库。

通过本项目产生的资产，可以一键投喂给 Google Gems、GPTs 或任何 RAG（检索增强生成）系统，从而生成一个具备“灵魂”的专家级数字孪生智能体。

---

## 🛠 核心功能描述

### 1. 原始语料标准化 (Standardized Raw Extraction)
系统通过 `extract_raw.py` 自动化处理分散在 `./documents` 文件夹中的多种格式素材。
- **全格式兼容**：支持 PDF（含扫描件识别引导）、DOCX、TXT 等。
- **智能纠码**：自动检测原始文件编码，解决中文环境下常见的乱码问题。
- **清洗降噪**：初步剔除文档中的非教学杂质（如广告页、页码信息）。

### 2. 原子化知识萃取 (Atomic Knowledge Alchemy)
这是本项目的核心价值，通过 `knowledge_factory.py` 执行，遵循严格的 `NLP_Protocol.md` 协议。
- **逻辑压缩 (Logic Compression)**：将臃肿的理论描述提炼为高密度的逻辑内核。
- **1:1 话术还原 (Verbatim Fidelity)**：对核心技术步骤进行无损复制，确保专家级严谨性。
- **提示词片段注入 (Prompt Fragments)**：为每个知识模块自动生成针对 LLM 调用场景的引导词，让 AI 知道何时、如何使用该知识。

### 3. 全局索引与资产管理 (Asset Indexing)
萃取后的结果存储在 `./output_kb` 目录，并配套生成 **`MASTER_INDEX.md`**：
- **模块化分类**：按功能（如：情绪管理、语言模式、亲子、婚恋）自动归类。
- **快速调用链接**：文件间建立逻辑索引，方便用户或 AI 快速定位特定模块。

## 🚀 项目优势 (Our Advantages)

1. **从“通用回复”到“硬核专家”**
   一般的 AI 智能体直接由原文训练，容易产生“幻觉”或给出肤浅的百科式回答。本项目通过**原子化预解析**，强制 AI 必须基于专家定义的“底层逻辑”进行决策，答复质量具备专业穿透力。

2. **高密度、低消耗**
   通过将 10 万+ 字的原始文本“脱水”压缩，我们提炼出了高强度的卡片式指令。这不仅极大降低了大模型上下文窗口的消耗，还显著提升了检索点位（RAG）的精准度。

3. **数字化人格克隆**
   不同于普通的问答助手，本项目不仅萃取“知识”，更捕获专家的“人格”。通过挂载特制的人格实操手册，让生成的智能体在语气、价值观（如三赢原则）、决策优先级上实现极致的 IP 还原。

4. **通用化平台潜力 (SaaS Ready)**
   系统的架构设计与李中莹 NLP 内容解耦。通过简单的协议更换，即可快速复刻到财富管理、法律咨询、企业内训等任何拥有专有知识沉淀的垂直领域。

## 💻 快速开始
1. **环境准备**：`uv sync`
2. **文本提取**：`uv run python extract_raw.py`
3. **知识炼成**：`uv run python knowledge_factory.py`
4. **投喂智能体**：将 `./output_kb` 文件夹下的 Markdown 文件作为“Knowledge”上传至 Google Gems 界面。
